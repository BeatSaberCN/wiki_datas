# 购买游戏


你可以在以下官方发售的平台买到节奏光剑的PC版本:
- [Steam平台](https://store.steampowered.com/app/620980/Beat_Saber/)
- [Oculus Rift商店](https://www.meta.com/zh-cn/experiences/pcvr/beat-saber/2448060205267927/)

你可以在以上任意平台购买游戏。这几乎不会影响到你后续的任何游戏体验。
