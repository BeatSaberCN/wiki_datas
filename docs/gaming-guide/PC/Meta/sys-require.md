# 系统需求

<提示>本页内容可能不可靠或存在个人偏见，请结合自身情况进行选购。</提示>

根据[Meta官方的说明](https://www.meta.com/help/quest/140991407990979/)，串流推荐的电脑配置为：

组件|推荐配置
---|---
CPU| Intel i7/AMD Ryzen7
显卡|Nvidia RTX 20系类/AMD Radeon RX 6000系列
内容|16GB DDR4 RAM
操作系统|Win10，Win11
USB端口|一个USB C端口

<提示>如果你的电脑不满足以下配置，并不代表无法游玩节奏光剑。你可能会遇到掉帧、卡顿、偶尔黑频、画面撕裂、延迟的问题。</提示>

## 内存配置

如果你要游玩社区自制的内容，16GB内存可能会无法满足需求。社区内容包含大量玩家自制谱面，这些谱面会占用你的系统内容。

## 显卡性能

<提示>此部分内容需要可能需要完善。</提示>

目前30/40系列中高端显卡性能已经足够应付大多数游戏，但是你需要**特别注意显存容量**。

游戏本身会占用约2GB显存，而Meta官方的串流软件也会占用2GB显存。因此如果你的显卡显存为4GB，则**刚好无法在官方有线串流中获得流畅体验**。

如果你想要启动SteamVR，则还会有约2GB的显存占用。此时总共需要6GB显存开销。如果你的显卡显存为6GB，则**硬件资源会非常紧张**。

节奏光剑不是一个显存开销很大的游戏，如果你还想在其他游戏中获得不错的体验，**请一定要选择大显存**的显卡。

## USB端口的稳定性

如果你要进行有线串流，你会需要一个高速率的Type C接口。

你可能会想买一根USB光纤线，它可以提升你的有线串流体验。

你需要一条具有5GB速率的USB 3.2数据线。

## 有线（无线）网卡

建议你的电脑配一个高速率的有线网卡，并通过它连接一性能不错的无线路由器，通过路由器连接头显。


为你的电脑配置一个无线网卡，可以用于解决Quest无法连接Meta服务的问题。这个无线网卡无需性能太好。

> 串流的视频码率通常在200Mbps或以下的水平，Quest2在低于80Mbps时能够获得较好的延迟效果。你可以参考这些信息，但你应该留足冗余，不要贴着需求买硬件。
