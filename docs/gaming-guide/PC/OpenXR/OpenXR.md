# 选择VR运行时环境

节奏光剑是通过OpenXR运行的，您需要安装合适的运行时环境，以驱动VR头显设备。

头显设备 | 必须安装 | 可选安装
---|---|---
Quest2、Quest3| [Meta Horizon Link](https://www.meta.com/help/quest/1517439565442928/) 或者 

## Quest头显设备
<提示 内容="此节内容适用于Quest 2、Quest 3、Quest3s等Meta推出的一体机设备。"/>

关于所需电脑的配置，可以参考[系统需求](Meta/sys-require)。如果你的电脑无法满足这个需求，请考虑使用一体机游戏方案。

有若干串流方案可供选择，你可以选择一个，或者全部安装：
- Meta官方提供的串流环境，请在电脑上安装[Meta Horizon Link](https://www.meta.com/help/quest/1517439565442928/)。
- Virtual Desktop。你需要在头显的应用商店里购买Virtual Desktop应用。然后在PC上安装[Streamer App](https://www.vrdesktop.net/)。

这两种方案中，你可以选择性地安装[SteamVR](https://store.steampowered.com/app/250820/SteamVR/)。节奏光剑不需要SteamVR，它会增加游戏延迟并消耗额外的显存资源，这不是必须的。但如果你想要使用SlimeVR等设备，那就需要安装。
