# 启动SteamVR

## 在串流环境下启动SteamVR

<警告 标题="不推荐使用Steam VR">大部分串流环境下都支持直接启动节奏光剑游戏，无需启动SteamVR。如果你要在串流环境下使用SteamVR，会引入额外的资源开销。</警告>

你可以在Virtual Desktop的菜单中找到启动SteamVR的按钮，点击以后就能启动。启动之后可以双击左手的菜单键进行桌面投屏的切换。

Oculus的有线串流和无线串流的环境下，你可以直接在Steam中启动SteamVR。但请注意如果想要通过SteamVR来运行节奏光剑，你需要将SteamVR设置为默认，或者使用适当的游戏启动参数。

## 在PCVR环境下启动SteamVR

直接启动即可。
