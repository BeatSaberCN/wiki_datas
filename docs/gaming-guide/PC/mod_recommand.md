# 模组推荐

你可以在[BeatMods](https://beatmods.com/mods)上找到可用的模组列表。

模组名称 | 介绍
---|---
[BeatLeader](/docs/mod-info/BeatLeader)和[ScoreSaber](/docs/mod-info/ScoreSaber) | 两个社区排行榜，能上传游戏录像。它们在安装后会自动绑定你的Steam账号。
[BetterSongSearch](https://beatmods.com/mods/225)| 游戏内搜索歌曲必备。
[PlaylistManager](https://beatmods.com/mods/170)| 游戏内下载歌单的模组。
[HRCounter](https://github.com/qe201020335/HRCounter)|是能够在游戏里显示心率的一个模组。这个模组暂时无法在BSManager上安装，需要手动安装。
[CustomBanners](https://beatmods.com/mods/217)|自定义旗子，Discord社区里面有很多玩家分享了他们的旗子。
