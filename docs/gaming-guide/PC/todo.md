# 上手清单

<提示 内容="本页清单适用于PC平台">这些内容需要你的游戏运行在PC上。如果你的设备是Quest一体机，且没有足够配置的PC电脑以及相关的串流硬件，请使用[Quest方案](../Quest/todo)</提示>

以下步骤列出了想要游玩PC平台游戏所需的完整步骤。

## 原版游戏
- [ ] [购买](buy_game)游戏。
- [ ] [安装VR设备](OpenXR)所需的环境。

## 模组安装
- [ ] 安装一个模组管理软件。你需要安装[BSManager](managers/BSManager)，然后再下载并安装游戏。
- [ ] 安装社区[自制谱面](songs)。

## 启动游戏

- [ ] 启动VR环境。如果电脑上有多个VR环境，还需要在想要使用的环境软件里，将其设置为默认环境。通常环境软件里会有提示。
- [ ] 启动游戏。你可以通过BSManager启动游戏，该软件可以为游戏增加启动参数，以指定游戏连接的VR环境。

## 进阶体验
- [ ] 安装[更多有趣的模组](mod_recommand)！
