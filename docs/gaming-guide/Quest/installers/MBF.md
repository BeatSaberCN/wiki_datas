# ModsBeforeFriday

[ModsBeforeFriday](https://mbf.bsquest.xyz/)是Lauriethefish维护的一个quest模组安装工具。

<注意 内容="ModsBeforeFriday需要Quest设备能够通过互联网下载模组。这些模组通常位于Github服务器，请确保服务可用。"/>

## 前置要求

### 开发者权限

Quest设备上登录的主账号应当具有Meta开发者权限。

### 外接设备需求

你需要用一台符合要求的电脑或者安卓手机打开ModsBeforeFriday的网页（见上文），并通过一根TypeC数据线连接Quest主机。

具体要求为：

你需要一个[支持WebUSB功能](https://developer.mozilla.org/en-US/docs/Web/API/WebUSB_API#browser_compatibility)的现代Chrome内核的浏览器。例如Edge和Chrome。不兼容Firefox和Safari浏览器。

## 使用方法

连接TypeC后直接在网页上连接Quest设备，然后在Quest上点击提示允许连接即可。

你可以在网页上按步骤完成游戏降级及模组安装。

<提示>每次MBF安装之后，都会自动将机器上过去的旧模组全部删除，模组的数据是安全的。你需要重新下载安装模组。</提示>

<提示 标题="如果游戏刚刚更新……">在游戏刚刚发布更新的一两天时间内，MBF会无法进行游戏降级。如果运气不好让你碰到了，就需要稍作等待，社区内的模组开发者会上传游戏差分文件，此时你就能正常使用了。</提示>

<提示 标题="游戏更新了，我的模组该怎么办……">在Quest平台上，各个模组开发者要为每一个游戏版本单独制作模组。大部分开发者选择**适配最新游戏版本**。偶尔有些开发者会为老版本游戏进行适配。这一现状导致旧版本游戏上的模组Bug永远不会得到修复。你应当在[mods.bsquest.xyz](https://mods.bsquest.xyz/)上检查最新的游戏版本支持哪些模组，并在适当时间进行更新。</提示>
